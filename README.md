# youtube-my-channel

YouTube Data API を使って、自分のチャンネルに関するデータを集計する

## Environment

```
$ sw_vers
ProductName:	Mac OS X
ProductVersion:	10.14.6
BuildVersion:	18G103

$ python --version

$ pip --version

```

## Setup

```
$ pip install -r requirements.txt
```
